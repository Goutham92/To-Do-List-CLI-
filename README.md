# To-Do List (CLI)

A simple To-Do List application built with Python for managing daily tasks through a command-line interface.

### Objective

Create a menu-driven task management application using Python that allows users to add, view, and delete tasks efficiently.

### Features

* Add Tasks
* Delete Tasks
* View Tasks
* Interactive Menu-Driven Interface
* Simple and User-Friendly CLI

### How to Run

1. Clone the repository:

```bash
git clone https://github.com/Goutham92/To-Do-List-CLI-.git
```

2. Navigate to the project directory:

```bash
cd To-Do-List-CLI-
```

3. Run the program:

```bash
python todo.py
```

4. Select an option from the menu.
