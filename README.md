# To-Do-List-CLI-
To-Do List application built with Python for managing daily tasks through a command-line interface.

## Synent Technologies – Python Development Internship Program

### Objective

Create a menu-driven task management application using Python.

### Features

* Add tasks
* Delete tasks
* View tasks
* Interactive menu-driven interface
* Simple and user-friendly CLI

### Technologies Used

* Python 3
* Visual Studio Code (VS Code)
* GitHub

### Project Structure

Task3-ToDo-List-CLI/
│
├── todo.py
└── README.md

### How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run:

python todo.py

4. Select an option from the menu.
